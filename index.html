MS-900: Understanding the Ecosystem

Exchange & Outlook

The backbone of enterprise communication and calendaring.

Microsoft Teams

The "Hub for Teamwork" - combining chat, meetings, and apps.

SharePoint & OneDrive

Intranets and personal cloud storage for content collaboration.

M365 Copilot

New Objective: AI-powered productivity across all apps.

Cloud Service Models

SaaS (M365)

  • Software as a Service
  • Zero server maintenance
  • Subscription-based usage

PaaS / IaaS

Infrastructure (VMs) and Platforms (DBs) where you manage more but own less hardware. (Contrast this with SaaS to explain M365 value).

Exam Tip: Know the difference between M365 Business (Small/Medium) and Enterprise E3/E5 (Corporate scale).

Shifting to The Administrator

MS-102 isn't about knowing what the tools are. It's about knowing how to deploy, secure, and govern them.

MS-102: Identity & Access

Entra ID (Azure AD)

  • Hybrid Sync: Using Entra Connect to bridge On-Prem and Cloud.
  • Conditional Access: The "If-Then" logic (e.g., If user is in Paris, Then require MFA).

Governance

  • PIM (Privileged Identity Management): Just-in-time access for Admins. No permanent "Global Admins".
  • SSPR: Self-Service Password Reset implementation.

MS-102: Defender & Protection

Defender for Office 365

Safe Links, Safe Attachments, and Anti-Phishing policies.

Defender for Endpoint

Onboarding Windows/iOS/Android devices to monitor for malware.

Defender for Cloud Apps

Shadow IT discovery - finding what apps your users are using without permission.

Microsoft Secure Score

The primary metric for measuring an organization's security posture. Admins must improve this score.

Microsoft Purview

Sensitivity Labels

"Classifying data as Public, Internal, or Highly Confidential."

Data Loss Prevention (DLP)

"Stopping credit card numbers from being emailed outside the company."

Retention Policies

"Ensuring data is kept for 7 years for legal reasons or deleted for privacy."
